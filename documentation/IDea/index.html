How to Update Bootstrap Framework?

IDea is a static HTML template so you need to update the Bootstrap framework manually, we highly recommend you check the repository https://getbootstrap.com/docs/3.3/ of bootstrap framework for new releases, and update it on your website if there are new releases.

In the template we keep all the files of bootstrap framework in a single folder html/bootstrap so that you can update it easily.

In order to update the bootstrap framework you need to download the latest version from its repository https://getbootstrap.com/docs/3.3/ or its dev branch https://github.com/twbs/bootstrap/tree/v3.4.0-dev and replace the files of html/bootstrap folder in the template, with the updated files.

Alternatively, you can add bootstrap framework in your website, via BootstrapCDN this way there is no need to download source files, you can read more on how you can add bootstrap framework via CDN on your website in the link that follows https://getbootstrap.com/docs/3.3/getting-started/

If you need assistance in order to update Bootstrap framework, contact us.

Below we give general instructions on how you can update a separate plugin inside the template.

For example let's update charts.js

  2. Next you need to find the folder of charts.js inside the template. That's easy because we keep all third party plugins inside a single folder: html/plugins so the charts.js file is located inside the html/plugins/charts folder.

How to Remove a plugin?

You can remove a plugin if you donâ€™t want the feature that is providing or if it is not maintained actively by its authors and you want to replace it with an other of your choice. In order to remove a plugin do the following:

General Guide
  1. Delete its folder from html/plugins folder.
  2. Delete related HTML markup from the related HTML files.

  3. Delete the css and js references from your HTML files e.g. for the owl-carousel plugin these:

    <link href="plugins/owl-carousel/assets/owl.carousel.css" rel="stylesheet">
<link href="plugins/owl-carousel/assets/owl.theme.default.min.css" rel="stylesheet">

    are the css references and this:

    <script src="plugins/owl-carousel/owl.carousel.js"></script>

    is the js reference.

  4. Delete its initialisation from the js/template.js file.
Example

For example, let's remove owl carousel plugin from the template

  1. First, we remove its folder from the html/plugins folder.

  2. Second, we remove related HTML markup. In order to find related markup we search for owl-carousel word inside the HTML file.

  3. Delete, the css and js references from your HTML files. In order to find related references we search for owl-carousel word inside the HTML file.

  4. Next, we should find its initialisation in html/js/template.js file and remove it. In order to find related initialisation we search for owl-carousel word inside the template.js file.

Customization & Styles

Skins

To change the default color scheme, please replace the red.css to the following tag in your head:

<link href="css/skins/red.css" rel="stylesheet">

Layout Style

The template has two different layout modes, Wide and Boxed. The wide one is the default option, if you want to use the Boxed version, add the boxed class to <body> tag as below:

<body class="boxed">

Background Patterns

There are 9 background patterns. Background patterns are available only in boxed layout mode. In order to apply a background pattern all you have to do is to add one of classes pattern-1 ... pattern-9 to body tag.

<body class="pattern-1 boxed">

Fonts

To change the embedded fonts, please take a look in the head part of the website and you will find these tags:

<link href="https://fonts.googleapis.com/css?family=Open+Sans:300,400,400i,700,700i&subset=latin-ext" rel="stylesheet" type="text/css">
<link href="https://fonts.googleapis.com/css?family=PT+Serif" rel="stylesheet" type="text/css">

To change the fonts, visit the https://fonts.google.com/ choose a font and use the generated code.

Fixed navigation positioning

By default the positioning of main navigation is set to fixed. To change this behavior delete the fixed class from the header.

<header class="header fixed clearfix">

Dark/Light footer

By default the footer is dark add the light class to #footer in order to change it from dark to light version.

<footer id="footer" class="light">

Dark/Light header

By default the header is light add the dark class to .header and .header-top divs in order to change them from light to dark versions.

<div class="header-top dark">
<header class="header fixed dark clearfix">

Transparent header

By default the header is not transparent add the transparent class to .header div in order to make the header transparent.

<header class="header transparent fixed clearfix">

Animations Effect

animate.css https://github.com/daneden/animate.css

You can add some Animations / Transitions to the elements which fires when an element scrolls into view. Great for emphasis, home pages, sliders, and general just-add-water-awesomeness.

Usage:

              <div class="object-non-visible" data-animation-effect="fadeInRight" data-effect-delay="200">
              ...
            </div>

The Attribute data-effect-delay is Optional and is set in Miliseconds.
